Быстрый старт

Основные шаги настройки

01

Скачать архив и распаковать mingw64

Папку mingw64 нужно перенести в корень диска, чтобы путь был C:\mingw64.

02

Перенести проект

Папку проекта лучше хранить по пути без русских букв. Например: C:\projects\cpp_start.

03

Добавить компилятор в Path

В переменные среды Windows добавить новый путь: C:\mingw64\bin.

04

Установить VS Code

После установки открыть проект и импортировать готовый профиль расширений.

05

Запустить тестовую программу

Нажать кнопку запуска в VS Code и проверить, что программа читает данные и создаёт результат.

Python

Python тоже можно настроить быстро

Для Python достаточно скачать установщик с официального сайта, запустить его и обязательно поставить галочку Add python.exe to PATH.

Главное правило

Если Python или C++ не запускается из терминала, чаще всего проблема в переменной Path или в пути к проекту.

FAQ

Частые проблемы

Почему лучше не использовать русские буквы в пути?

Из-за кодировок и особенностей отладчика VS Code пути с кириллицей могут вызывать странные ошибки.

Что делать, если breakpoints игнорируются?

Попробуйте упростить файл, убрать лишний код, создать новую папку и скопировать туда настроенную папку .vscode.

Что делать, если пропала кнопка запуска?

Проверьте расширения VS Code или создайте новый профиль расширений.

Что делать, если переменные не видны в отладке?

Можно добавить нужную переменную в Watch через контекстное меню в режиме отладки.

Готов начать?

Открой репозиторий, скачай файлы и пройди настройку по шагам.
