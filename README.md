## Problem with Russian letters

For those curious why the author dislikes them so much?!
It‚Äôs all about UTF-8 encoding. To use Russian characters and keep things more or less stable, you need to enable the beta version of support for all world languages ‚Äî but some programs require this to be turned off.

Conclusion: It‚Äôs easier not to enable this and just make everything in English.

> _Seriously: avoid Russian characters. They break half of the universe and the VS Code debugger at the same time :)_
